# SmartFit - Your Smart Fashion Assistant

SmartFit is a web-based fashion assistant that helps you organize your wardrobe, get personalized outfit suggestions, and make smarter clothing decisions using AI.

### Virtual Wardrobe
- **Photo Upload**: Drag-and-drop or browse to upload photos of your clothes
- **AI Categorization**: Automatically detects and categorizes items (tops, pants, shoes, accessories)
- **Category Filtering**: Easy navigation through your wardrobe by category
- **Item Management**: Track all your clothing items in one place

### Smart Outfit Recommendations
- **Daily Suggestions**: Get personalized outfit recommendations based on:
  - Real-time weather data
  - Your personal style preferences
  - Occasion (casual, formal, sporty, evening)
- **Shuffle Feature**: Generate new outfit combinations instantly
- **Save Outfits**: Keep your favorite outfit combinations

### Sustainability Focus
- **Reuse Existing Clothes**: Encourages using what you already own
- **Reduce Purchases**: Helps you make smarter clothing decisions
- **Partner Store Suggestions**: Optional affiliate links for matching items (Premium)

### Profile & Settings
- **Style Preferences**: Customize your fashion style (casual, formal, sporty, elegant, minimalist, colorful)
- **Location Settings**: Set your city for accurate weather-based recommendations
- **Sustainability Mode**: Toggle features that promote eco-friendly fashion choices

### Premium Features
- **Free Plan**:
  - Upload up to 50 items
  - 5 outfit suggestions per day
  - Basic weather integration
  - Manual categorization

- **Premium Plan** ($9.99/month):
  - Unlimited wardrobe items
  - Unlimited outfit combinations
  - Advanced AI style matching
  - AI-powered categorization
  - Calendar event integration
  - Sustainability insights
  - Partner store suggestions

## ğŸš€ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No server-side dependencies required

### Installation

1. Clone the repository:
```bash
git clone https://github.com/linobr/SmartFit.git
cd SmartFit
```

2. Open `index.html` in your web browser:
```bash
# Using Python's built-in server
python3 -m http.server 8000

# Or using Node.js
npx http-server

# Then open http://localhost:8000 in your browser
```

Alternatively, simply open `index.html` directly in your browser.
